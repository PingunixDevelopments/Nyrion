# Nyrion
What is **Nyrion**?
Nyrion is a CLI build for people that want a simple way of doing things. Nyrion basically holds your hand into the CLI world. But, it also is made for power users.

Features
- Easy file managing
- Disk management
- Encoding
- Network information

**How to install?**

- Download the .zip file and save it somewhere
- Download .Net 4.0
- Install .Net 4.0
- Unzip Nyrion
- Go into the bin folder and double click the exe
- Enjoy Nyrion.
