# Nyrion
What is **Nyrion**?
Nyrion is a CLI build for people that want a simple way of doing things. Nyrion basically holds your hand into the CLI world. But, it also is made for power users.

Features
- Easy file managing
- Disk management
- Encoding
- Network information
